04 JWT 디코더

Unix 타임스탬프 ↔ 사람이 읽는 날짜 형식을 변환합니다.

왜 DevToolKit인가요?

DevToolKit은 회원가입 없이 즉시 사용 가능한 개발자 도구 모음입니다. 모든 처리는 사용자의 브라우저 안에서 수행되며, 입력값이 외부 서버로 전송되지 않아 개인정보와 민감 데이터를 안전하게 다룰 수 있습니다. JSON, Base64, URL, JWT, UUID, Hash, Timestamp 등 자주 쓰는 기능을 한 페이지에서 빠르게 처리하세요.

자주 묻는 질문 (FAQ)

DevToolKit은 무료인가요?

네. 모든 도구는 무료이며, 회원가입이나 API 키 없이 즉시 사용할 수 있습니다.

입력한 데이터가 외부 서버로 전송되나요?

아닙니다. JSON, Base64, JWT, 해시 등 모든 처리는 사용자의 브라우저 안에서 JavaScript와 Web Crypto API로 수행됩니다. 네트워크 탭에서 직접 확인하실 수 있습니다.

JWT 디코더가 서명도 검증하나요?

현재 버전은 Header와 Payload의 디코딩만 제공합니다. 서명 검증은 발급 측 secret 또는 public key가 필요하므로 제공하지 않습니다.

왜 MD5는 없나요?

MD5는 충돌 공격이 알려져 있어 보안 목적에는 권장되지 않습니다. Web Crypto API의 표준 알고리즘인 SHA-1 / SHA-256 / SHA-512만 제공합니다.

오프라인에서도 동작하나요?

네. 한 번 페이지를 연 뒤에는 정적 자산이 브라우저에 캐시되어 네트워크 없이도 모든 도구가 동작합니다.

새 도구를 요청하고 싶어요.

GitHub 저장소의 Issues 또는 이메일로 문의 부탁드립니다. 향후 Color picker, Cron 파서, JSON↔YAML 변환 등을 추가할 예정입니다.
